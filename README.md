# Stress Management

A web-based stress management platform designed to help users improve their mental well-being through relaxation activities, mindfulness exercises, and self-care resources.

## Features

- User Registration and Login
- Guided Meditation Section
- Breathing Exercises
- Relaxing Music Player
- Drawing Activity for Stress Relief
- Photo Upload and Gallery
- Contact Form
- Feedback System
- Responsive User Interface

## Technologies Used

- PHP
- MySQL
- HTML5
- CSS3
- JavaScript

## Installation

1. Clone the repository

```bash
git clone https://github.com/Nidhiyadav2831/stress-management.git
```

2. Move the project to your local server directory (XAMPP/WAMP/MAMP).

3. Create a MySQL database.

4. Import the SQL file:

```text
database/setup.sql
```

5. Update database credentials in:

```text
includes/db_connect.php
```

6. Start Apache and MySQL.

7. Open in browser:

```text
http://localhost/stress-management
```
